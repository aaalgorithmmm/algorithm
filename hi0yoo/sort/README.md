# 정렬 알고리즘

## 버블 정렬
- 데이터의 인접 요소끼리 비교하여 swap 연산을 통해 정렬
- 시간복잡도 O(n^2)
- 오름차순 정렬 기준, 처음 요소부터 마지막 요소까지 반복
  - 선택 요소와 다음 요소를 비교하여 선택한 요소가 더 크면 swap
  - 다음 요소로 이동


## 선택 정렬
- 선택한 요소 이전은 모두 정렬되어 있다고 가정
- 선택 요소 이후 요소들 중, 최솟값 또는 최대값을 찾아 선택 요소와 swap
- 시간복잡도 O(n^2)


## 삽입 정렬
- 선택한 요소를 정렬된 범위의 적절한 위치에 삽입하는 방식
- 시간복잡도 O(n^2)

```text
[42, 32, 24, 60, 40]

(1) 1번 요소(42) 선택. 1~1 이내 범위에서 적절한 위치 1
    [42, 32, 24, 60, 40]
(2) 2번 요소(32) 선택. 1~2 이내 범위에서 적절한 위치 1. 1 위치에 2번요소 삽입
    [32, 42, 24, 60, 40] 
(3) 3번 요소(24) 선택. 1~3 이내 범위에서 적절한 위치 1. 1 위치에 3번요소 삽입
    [24, 32, 42, 60, 40]
(4) 4번 요소(60) 선택. 1~4 이내 범위에서 적절한 위치 4. 4 위치에 4번요소 삽입
    [24, 32, 42, 60, 40]
(5) 5번 요소(40) 선택. 1~4 이내 범위에서 적절한 위치 3. 3 위치에 4번요소 삽입
    [24, 32, 40, 42, 60]
```


## 퀵 정렬
- 기준값을 선택하여 기준값보다 큰값, 작은값으로 분류하는 것을 반복하여 정렬 수행
  - 기준값을 기준으로 분리집합이 생성되는데, 각 분리집합에 대해 동일하게 정렬을 수행
  - 분리집합이 1개 이하가 될 때 까지 각 분리집합에 정렬을 수행해야 한다.
- 기준값 선정에 따라 시간복잡도에 많은 영향
- 평균 시간복잡도는 O(nlogn)


## 병합 정렬
- 분할 정복 방식을 사용하여 데이터를 분할하고 분할한 집합을 정렬하며 합친다.
- 평균 시간복잡도는 O(nlogn)


## 기수 정렬
- 값을 비교할 자릿수를 정하여 해당 자릿수만 비교하여 정렬한다.
- 1자리 -> 10자리 -> 100자리 -> ...
- 큐를 이용하여 각 자릿수를 입력 및 정렬
- 마지막 자릿수를 기준으로 정렬할 때까지 반복
- 시간 복잡도 O(kn) : k 는 자릿수
